# 用户 User 埋点

| 埋点类型 type | 说明                     |
| ------------- | ------------------------ |
| user_register | 用户注册数（每日累计）   |
| user_active   | 活跃用户数（每日/周/月） |

> 说明：活跃用户可通过登录、发帖、评论、浏览等行为判定。

- [x] **粉丝数/关注数**

  - 字段：无直接字段，需通过 UserFollow 表统计
  - 触发接口：POST `/user/:id/follow`，DELETE `/user/:id/follow`
  - 埋点方式：关注时插入 UserFollow，取关时删除
  - 注意事项：可通过聚合统计

- [] **用户活跃度**
  - 字段：无直接字段，需通过行为日志统计
  - 触发接口：发日记、评论、点赞、收藏等
  - 埋点方式：可通过各行为表统计
  - 注意事项：如需单独字段可后续补充
